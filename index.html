Simple Game+

A remastered version of the original Simple Game created by ChatGPT and DALL-E on January 10th, 2022.

The original game was curated in Flashpoint Archive on March 18th, 2023.

Original note: "The HTML, JavaScript, CSS and images were all generated using OpenAI. All scripting was generated by ChatGPT, and images were created using DALL-E. The only human input was the initial prompt for ChatGPT ('Write HTML and JavaScript for a simple video game.'), and the prompts for the art ('Pixel Art of a Character' and 'Pixel Art of a House')."

This remaster was created in 2025 using Trae with Claude 3.7 Sonnet.
